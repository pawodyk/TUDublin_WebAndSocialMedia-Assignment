GetIT UK

This page is an interactive website for job-seeking IT professionals. It utilizes the ©Adzuna API to provide the user with the most up to date jobs in United Kingdom.

This site also uses ©Bing Maps API to display available jobs on the map. The map is interactive and once the particular cluser on the map is clicked it will filter results to jobs in this particular location.
